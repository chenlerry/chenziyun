思路
- 设“臂厚”为 a ，凹进去的台阶（每边伸出的长度）为 b 。
- 该十字形可看作两个长方形的并集：竖条 a × (a + 2b) 与横条 a × (a + 2b) ，中间重叠一个正方形 a × a 。
- 面积公式： S = 2*a*(a + 2*b) - a*a = a*(a + 4*b) 。
- 根据图中标注，取 a = 4 ， b = 2 ，则 S = 48 。

```c
#include <iostream>
using namespace std;
int main() {
    int a = 4; // 臂厚
    int b = 2; // 台阶长度
    int area = 2 * a * (a + 2 * b) - a * a; // a*(a+4b)

    cout << "面积 S = " << area << std::endl; // 结果：48
    return 0;
}
```
