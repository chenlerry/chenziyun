思路
- 设“臂厚”为 a ，凹进去的台阶（每边伸出的长度）为 b 。
- 该十字形可看作两个长方形的并集：竖条 a × (a + 2b) 与横条 a × (a + 2b) ，中间重叠一个正方形 a × a 。
- 面积公式： S = 2*a*(a + 2*b) - a*a = a*(a + 4*b) 。
- 根据图中标注，取 a = 4 ， b = 2 ，则 S = 48 。


```c
#include <iostream>
using namespace std;
int main() {
    int a = 4; // 臂厚
    int b = 2; // 台阶长度
    int area = 2 * a * (a + 2 * b) - a * a; // a*(a+4b)

    cout << "面积 S = " << area << std::endl; // 结果：48
    return 0;
}
```

简单思路一
- 大正方形减去四个角
- 大正方形边长： 4 + 2 + 2 = 8
- 面积： 8 × 8 = 64
- 四个角都是小正方形，边长 2 ，面积每个 2 × 2 = 4 ，总共 4 × 4 = 16
- 剩余面积： 64 − 16 = 48

简单思路二
- 中间正方形加四个长方形
- 中间正方形： 4 × 4 = 16
- 四个长方形：每个 2 × 4 = 8 ，总共 4 × 8 = 32
- 总面积： 16 + 32 = 48

```c
#include <iostream>
using namespace std;

int main() {
    int a = 4; // 臂的厚度
    int b = 2; // 每边伸出的长度
    int bigSquare = (a + 2*b) * (a + 2*b);
    int corners = 4 * b * b;
    int area = bigSquare - corners;

    cout << "面积 S = " << area << endl; // 48
    return 0;
}
```
